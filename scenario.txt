1. Пользователь регистрируется
  - Создается запись в коллекции users
  - Входные параметры: login (логин), password (пароль)
2. Пользователь авторизуется
  - Пользователь получает данные:
    - о пользователе;
    - список его СД (с пометкой 'только просмотр' при соответсвующем доступе);
    - список сообщений с измнениями в доступных ему СД;
    - СД, назначенное по умолчанию;
3. Пользователь создает СД 
  - Создается запись в коллекции trees
  - Создается запись в коллекции permissions
  - Созданное СД становится 'по умолчанию' для этого пользователя
  - Входные параметры: tree_name (наименование СД)
4. Пользователь дает доступ к СД другому пользователю
  - Создается запись в коллекции permissions
  - Входные параметры: user_id (идентификатор пользователя), permission_type (тип доступа)
